# Pettomets

A platform for connecting pet owners with trusted local pet service providers.

## About

Pettomets is a web-based platform that helps pet owners find reliable pet care services in their area, including grooming, training, walking, and boarding services. Service providers can also register to offer their services to the community.

## Features

- **Service Discovery**: Find trusted pet service providers near you
- **Multiple Services**: Access to grooming, training, walking, and boarding services
- **Easy Registration**: Simple onboarding process for service providers
- **User-Friendly Interface**: Clean and intuitive design for easy navigation

## Website Structure

- `index.html` - Home page with main service information
- `about.html` - About page describing the platform
- `contact.html` - Contact form for inquiries
- `join.html` - Service provider onboarding form
- `terms.html` - Terms of Service
- `privacy.html` - Privacy Policy
- `style.css` - Main stylesheet
- `script.js` - JavaScript functionality
- `assets/` - Images and icons

## How to Use

### Viewing Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/SumitMarandi/demo.git
   ```

2. Open `index.html` in your web browser

### Deployment

This is a static website that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## GitHub Pages Deployment

To publish this site using GitHub Pages:

1. Go to repository Settings
2. Navigate to "Pages" section
3. Under "Source", select the main branch
4. Click "Save"
5. The site will be available at `https://sumitmarandi.github.io/demo/`
